## 🌟 Project Overview

**WoolFlow** is a modern minimalist e-commerce UI designed for sustainable wool sneakers. The interface focuses on clean product discovery, curated collections, and a smooth shopping experience inspired by premium footwear brands like Allbirds, Nike, and Adidas.

Built with pure **HTML5**, **CSS3**, and **vanilla JavaScript**, this project demonstrates how to create a production-ready e-commerce frontend without heavy frameworks — perfect for learning modern web development fundamentals.

### ✨ Design Philosophy

- **Clean & Minimalist** — Focus on products, not clutter
- **Sustainable Aesthetic** — Earth tones and natural inspiration
- **Smooth Interactions** — Delightful micro-animations
- **Mobile-First** — Responsive across all devices

## 🚀 Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (recommended for best experience)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/nareshchandu17/woolflow-ecommerce.git
   cd woolflow-ecommerce
   ```

2. **Start a local server**
   
   Using Python:
   ```bash
   python -m http.server 8080
   ```
   
   Using Node.js (http-server):
   ```bash
   npx http-server -p 8080
   ```
   
   Using PHP:
   ```bash
   php -S localhost:8080
   ```

3. **Open in browser**
   
   Navigate to `http://localhost:8080`
